acwing-785. 快速排序

本文最后更新于：2022年7月25日 上午

给定你一个长度为 n 的整数数列。

请你使用快速排序对这个数列按照从小到大进行排序。

并将排好序的数列按顺序输出。

输入格式
输入共两行，第一行包含整数 n。

第二行包含 n 个整数（所有整数均在 $1∼10^9$ 范围内），表示整个数列。

输出格式
输出共一行，包含 n 个整数，表示排好序的数列。

数据范围
1≤n≤100000

输入样例：
5
3 1 2 4 5
输出样例：
1 2 3 4 5

2022 7 25二刷

需要注意的：
1、递归返回条件；
2、do while 的条件；
3、if的条件；
4、递归的参数

快排模板quick_sort，熟记

注意：
do i++; while(q[i] < x); // **从小到大排序，左边都是 <= x的**
do j--; while(q[j] > x); // 从小到大排序，右边都是 >= x的
